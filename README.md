# CIVE_202_Project-3
This repository contains the files and information for CIVE 202 Project #3. The Python code in this project is used to visualize transportation data trends with different charts. Since transportation systems are important to everyday life two datasets were used to better understand transportation: the National Household Travel Survey (NHTS) and the Next Generation Simulation (NGSIM) data. These datasets were analyzed to look at traffic patterns and behaviors, and the results were shown using a bar chart, histogram, boxplot, and two different time-series plots that are displayed in the code file project_3.ipynb. 
# How To Run
Download the Python Notebook to your device. 
Make sure you also downloaded the NHTS.csv and NGSIM.csv files and put them and the Python file in the same file on your device. 
Run each cell from top to bottom. The notebook is seperated and labeled to make this simple. 
  Section one: being the Package imports 
  Section two: being to load and check the datsets 
  Section three: The NHTS visualization (bar charts, histograms, boxplots)
  Section four: The NGSIM time-series plots (speed, acceleration, position, gap distance)
  Section five: IDM simulation(function definition, EUler time stepping, comparision plots)
# User guide for Prompts
This Python notebook does not require any user inputs all variables are sekected and the IDM parameters are set directly into the code cells.
To change the trajectory pair you must modifiy the trajectopry_number variable in section 4.
