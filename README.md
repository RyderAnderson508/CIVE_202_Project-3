# CIVE_202_Project-3
This repository contains the files and information for CIVE 202 Project #3. The Python code in this project is used to visualize transportation data trends with different charts. Since transportation systems are important to everyday life and the U.S. economy, two datasets were used to better understand transportation: the National Household Travel Survey (NHTS) and the Next Generation Simulation (NGSIM) data. These datasets were analyzed to look at traffic patterns and behaviors, and the results were shown using a bar chart, histogram, boxplot, and two different time-series plots that are displayed in the code file project_3.ipynb.
